# 🏥 iNurseApp Frontend

Sistema de Gestão para Profissionais de Enfermagem

## 🚀 Deploy Automático

Para fazer deploy do projeto para o GitHub e Netlify, use o script automatizado:

```powershell
.\deploy-to-github.ps1
```

Ou veja a documentação completa em `DEPLOY-AUTOMATICO.md`

## 📦 Estrutura do Projeto

```
inurseapp-frontend/
├── index.html              # Página principal
├── styles.css              # Estilos CSS
├── script.js               # JavaScript
├── deploy-to-github.ps1    # Script de deploy automático
├── DEPLOY-AUTOMATICO.md    # Documentação de deploy
├── QUICK-DEPLOY.md         # Guia rápido de deploy
└── README.md               # Este arquivo
```

## 📝 Desenvolvimento

Para desenvolver localmente, basta abrir o `index.html` em um navegador ou usar um servidor local:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```
