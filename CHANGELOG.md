# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-02-11

### ✨ Adicionado
- Landing page responsiva com hero section animado
- Sistema de autenticação (login e registro)
- Integração completa com backend API
- Google Analytics (G-5F3XF8G5KN)
- Página de testes da API (test.html)
- Página 404 personalizada
- SEO meta tags (Open Graph, Twitter Cards)
- Manifest.json para PWA
- Robots.txt para SEO
- Sitemap.xml
- Sistema de notificações toast
- Loading states e spinners
- Headers de segurança via Netlify
- Redirects automáticos (HTTP → HTTPS, www → non-www)

### 🎨 Design
- Design system com variáveis CSS
- Paleta de cores profissional
- Tipografia responsiva
- Animações suaves
- Cards com hover effects
- Waveform animation no hero
- Gradient text effects

### 🔒 Segurança
- Headers de segurança (X-Frame-Options, CSP, etc)
- JWT token em localStorage
- Validação de formulários
- Password strength requirements
- Auto-logout em 401

### 📱 Responsividade
- Mobile-first approach
- Breakpoints otimizados
- Navigation adaptativa
- Grid layouts responsivos

### 🚀 Performance
- CSS/JS minificação via Netlify
- Cache headers otimizados
- Lazy loading preparado
- CDN global via Netlify
